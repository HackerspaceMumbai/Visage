# Product Overview

Visage is a comprehensive meetup management platform designed for the largest Open Source Software community in Mumbai with 10+ years of tech meetup experience.

## Core Purpose

Solves three critical problems for large-scale tech meetups:

1. **Smart Registration Curation** - AI-powered attendee selection based on event themes while ensuring diversity and inclusivity
2. **Real-time Check-in/Check-out** - Session-level attendance tracking for experience optimization and building code compliance
3. **DPDP Compliance** - Secure data handling within Indian jurisdiction, adhering to India's Digital Personal Data Protection Act (equivalent to GDPR)

## Key Features

- Event registration and attendee management
- Photo upload and management via Cloudinary integration
- AI-powered attendee curation using Azure OpenAI
- Real-time attendance tracking
- Multi-platform support (Web, Mobile, Desktop)
- Comprehensive analytics and monitoring

## Target Users

- Event organizers managing large tech meetups
- Community managers requiring attendee insights
- Venue managers needing compliance reporting
- Attendees seeking streamlined event experiences
